# Problem: Count Subsequence Pairs with Equal GCD

## Problem Statement
You are given an array `nums`.  
Count the number of ways to split elements into two subsequences such that the **GCD of both subsequences is equal**.  
Return the answer modulo `1e9+7`.

---

## Sample Input
nums = [2, 4, 6]

## Sample Output
4

## Explanation
There are 4 valid ways to form subsequence pairs where both subsequences have the same GCD.
