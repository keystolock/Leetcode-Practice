# Problem: Sum and Multiply Digits (Ignoring Zeros)

## Description
Given an integer `n`, construct a new number `x` by removing all zeros from `n`.  
Also, compute the sum `s` of all non-zero digits in `n`.  
Finally, return the product `x * s`.

---

## Example

### Input
n = 1023

### Output
738

### Explanation
- Digits: `1, 0, 2, 3`
- Remove zeros → `x = 123`
- Sum of non-zero digits → `s = 1 + 2 + 3 = 6`
- Result → `123 * 6 = 738`

---

## Constraints
- \(1 \leq n \leq 10^9\)
