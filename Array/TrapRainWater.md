# Trapping Rain Water - LeetCode Solution

## 📌 Overview
This repository contains my solution to the classic **LeetCode Problem #42: Trapping Rain Water**.  
The problem challenges us to calculate how much water can be trapped between bars in an elevation map.

## 🚀 Approach
- Implemented using the **two-pointer technique**.
- Time Complexity: **O(n)**
- Space Complexity: **O(1)**
- Handles edge cases (empty array, strictly ascending/descending arrays).

## 🧩 Example
Input:[4,2,0,3,2,5]
Output: 9


## 📊 Complexity Analysis

Time: O(n) — single pass through the array.
Space: O(1) — constant extra space.
