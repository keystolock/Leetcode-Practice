# Partition of an Array according to the given pivot

# Problem:

You are given an integer array nums and a piviot element . Rearrange the nums array such that:

 1. Every element less than  pivot comes before  pivot
 2. Every element greater than pivot comes after pivot
 3. Elements equal to pivot comes in between less and greater elements


Example 1:

Input: nums = [9,12,5,10,14,3,10], pivot = 10
Output: [9,5,3,10,10,12,14]

# Constraints:
pivot equals to an element of nums.
